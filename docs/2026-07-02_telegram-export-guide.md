# Экспорт постов из Telegram

## Зачем

Telegram-канал «Портфельный инвестор» — основной источник контента. Чтобы перенести статьи на сайт, нужен машиночитаемый экспорт постов: дата, текст, изображения, ссылки.

---

## Шаг 1. Открыть Telegram Desktop

1. Установить [Telegram Desktop](https://desktop.telegram.org/), если ещё не установлен.
2. Войти в аккаунт, который владеет или администрирует канал «Портфельный инвестор».

## Шаг 2. Запустить экспорт

1. Открыть канал.
2. Нажать на **название канала** сверху → откроется информация.
3. В меню (⋮) выбрать **Экспорт данных канала** / **Export channel data**.
4. В открывшемся окне настроить:
   - **Тип чата:** только канал.
   - **Формат:** JSON (обязательно).
   - **Содержимое:**
     - ✅ Текстовые сообщения
     - ✅ Фото
     - ✅ Видео (опционально, тяжёлые)
     - ✅ Файлы (опционально)
   - **Диапазон дат:** с самого начала канала до сегодня.
5. Нажать **Экспортировать** и дождаться окончания. В зависимости от объёма — от минут до часа.

## Шаг 3. Полученные файлы

Telegram создаст папку вида:

```
ChatExport_YYYY-MM-DD_канал/
├── result.json          # основной файл: дата, текст, id, медиа
├── photos/              # папка с фото
├── files/               # документы
└── video_files/         # видео
```

Нужен именно `result.json` и папка `photos/`.

## Шаг 4. Перенос в проект

1. Скопировать `result.json` в папку проекта:
   ```
   Site/telegram-export/result.json
   ```
2. Скопировать папку `photos/` в:
   ```
   Site/telegram-export/photos/
   ```
3. Запустить скрипт конвертации:
   ```bash
   cd Site
   npm run convert-telegram
   ```

Скрипт создаст:

- `posts/YYYY-MM-DD_[slug]_post.json` для каждого поста;
- скопирует изображения в `public/images/YYYY-MM-DD_[topic]_hero.[ext]`;
- обновит индекс постов.

## Шаг 5. Проверка

После конвертации выполнить:

```bash
npm run check
npm run build
```

Если всё проходит — `git add -A && git commit -m "content: import Telegram posts" && git push`.

---

## Ограничения

- Telegram не экспортирует опросы, реакции, статистику просмотров.
- Посты с несколькими фото — используется только первое как hero-изображение.
- Длинные посты с серией сообщений экспортируются как отдельные сообщения. Если пост был разбит на несколько частей, их нужно объединить вручную после конвертации.

---

## Альтернатива: ручной импорт

Если экспорт Telegram недоступен, можно:

1. Скопировать текст поста вручную.
2. Создать файл `posts/YYYY-MM-DD_[slug]_post.json` по образцу `posts/2026-06-21_future-of-agriculture_post.json`.
3. Добавить изображение в `public/images/`.
4. Запустить `npm run build`.
